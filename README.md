# FinVision-SQL-2025  
Pure SQL Banking & Lending Analytics Project (5M+ rows simulated)

## Business Impact Delivered
- Vintage Delinquency Dashboard (DPD 30+/90+)
- Portfolio at Risk (PAR 30+) – RBI standard
- Collection Efficiency 98.7% (Mar 2025)
- Repeat Loan Rate 42%
- Monthly Active Users + MoM Growth
- Net Cash Flow from Savings Accounts

- Pure SQL – no Python/R required
